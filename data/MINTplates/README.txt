MINTcodes "tRF license-plates"
------------------------------

1. General Information
----------------------
This code was created by Venetia Pliatsika and Isidore Rigoutsos.
It can be used to create "tRF license-plates" from sequences and to decode "tRF license-plates" to
sequences.
For more information on "tRF license-plates" and tRFs, visit https://cm.jefferson.edu/MINTbase
Contact us at: https://cm.jefferson.edu/contact-us/

3. Usage Information
--------------------
Usage:
	java MINTcodes_tRF_license_plates -e example_sequences_to_encode.txt
	java MINTcodes_tRF_license_plates -d example_license_plates_to_decode.txt

Replace the files above with your files!
